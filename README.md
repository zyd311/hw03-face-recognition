# HW03 人脸识别系统
基于 face_recognition 和 Streamlit 实现的人脸检测 Web 应用。

## 功能说明
1.  支持上传图片，自动检测人脸（支持正面/侧面）
2.  用绿色框标出人脸位置
3.  显示检测到的人脸数量

## 运行说明
1.  安装依赖：
    pip install -r requirements.txt

2.  启动应用：
    streamlit run app.py

3.  访问地址：
    http://localhost:8501
